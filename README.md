# library_system_ui
A complete Library Management System in python
Here’s a solid starting point for a Library Management System in Python with a modern UI using customtkinter. It lets you add books, view them, search, and track availability. Later, you can extend it with features like borrowing/returning, user accounts, and persistent storage (JSON/SQLite).


✨ Features
Add books with title, author, year.

Search books by title or author.

Borrow and return books (status changes between Available and Borrowed).

Modern dark UI with customtkinter.

Displays all books in a scrollable list.

User management: Add members with name + email.

Borrowing system: Books are linked to specific users when borrowed.

Return system: Clears borrower info when returned.

Persistent storage: All data saved in SQLite (library.db).

Modern UI with customtkinter.
